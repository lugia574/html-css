HTML- 블록 레벨 요소와 인라인 요소

1. 블록 요소
    1. DIV, H1, P
    2. 사용 가능한 최대 가로 너비를 사용한다.
    3. 크기를 지정할 수 있다.
    4. (widith: 100%; height 0; 으로 시작)
    5. 수직으로 쌓임
    6. margin, padding 위,아래,좌,우 모두 가능함
    7. 레이아웃 용도

2. 인라인 요소
    1.SPAN, IMG
    2. 필요한 만큼의 너비를 사용한다.
    3. 크기를 지정할 수 없다.
    4. (widith: 0; height 0; 으로 시작)
    5. 수평으로 쌓임 / 줄바꿈하면 뛰어쓰기가 됨
    6. margin, padding 위,아래 사용할 수 없다
    7. 텍스트 용도


* 우선 순위결정
    같은 요소가 여런선언의 대상이될 경우
    어떤 선언의 CSS 속성(property)을 우선 적용할지 결정하는 방법

    1. 명시도 점수가 높은 선언이 우선(명시도)
    2. 점수가 같은 경우. 가장 마지막에 해석되는 선언이 우선 (우선 순위)
    3. 상속이 직접적으로 주는것보다 떨어짐
    4. !important 가 적용된 선언방식이 젤 먼저
    상속(-) < 전체(0) < 태그(1) < 클래스(10) < 아이디(100) <
